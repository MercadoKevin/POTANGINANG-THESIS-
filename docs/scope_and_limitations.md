# Scope and Limitations

## Scope

The prototype focuses on:
- real-time frame acquisition from a webcam
- live detection pipeline demonstration
- suspicious-item screening from X-ray monitor imagery
- modular design for future CNN integration

## Limitations

- The current demo mode uses heuristic analysis rather than a finalized trained CNN.
- Accuracy claims cannot yet be made without formal dataset training and testing.
- Webcam capture may introduce glare, perspective distortion, and variable lighting.
- Direct integration with the X-ray machine is outside the current scope due to access restrictions.

## Research Value

Even at this stage, the system demonstrates the feasibility of a non-invasive, real-time monitoring assistant for X-ray screening environments.
